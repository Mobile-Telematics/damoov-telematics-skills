# Damoov Plugin

Self-hosted plugin package for Codex, Claude Code, and Cursor.

Official site: https://damoov.com

Documentation: https://docs.damoov.com

Source repository: https://github.com/Mobile-Telematics/damoov-telematics-skills

Use the orchestrator skill first:

```text
Use $damoov-telematics-skill to route this Damoov integration task.
```

For Claude Code, plugin skills are namespaced after installation:

```text
/damoov-telematics:damoov-telematics-skill
```

This plugin is generated from the public Damoov skills:

- `damoov-telematics-skill`
- Android, iOS, Flutter, and React Native SDK integration skills
- backend registration skill
- user management skill
- product management skill
- trips and statistics skill
- integration testing skill

The published plugin package is self-contained. It does not require files outside the plugin
directory after installation.
